# HEX - your meta browser

HEX is the product identity for the project formerly called Browser Base.

## Core Meaning

HEX compresses several ideas into one word:

- hexagon
- honeycomb
- bee geometry
- modular structure
- interconnected cells
- hexadecimal systems
- symbolic compression

## Product Idea

HEX is a local-first Windows desktop companion that floats above the user's normal work.

It is not a browser replacement. It is a calm continuity layer across applications, websites, tabs, and files:

- threads preserve intent
- saved links keep relevant pages attached to the thread
- quick notes and Next Up preserve context
- pins preserve the understanding accumulated while working
- thread folders keep local materials attached to the work
- an edge overlay keeps HEX available without becoming another destination

The primary experience is a small Windows edge tab that expands into a compact thread, link, and file holder. The larger workspace remains available for deeper management.

## Mascot Meaning

The bee is no longer decoration. It represents the intelligence moving through the system:

- explorer
- collector
- pollinator
- builder of hexagonal structure

The bee moves through the web. HEX is the comb it builds.

## Design Direction

HEX should feel calm, geometric, warm, lightweight, operational, and memorable.

Avoid cyberpunk terminal aesthetics, noisy dashboards, fake futuristic styling, and AI novelty clutter.
