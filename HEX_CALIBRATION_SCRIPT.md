# HEX Calibration Script

Read this aloud when explaining HEX or deciding what belongs in the product.

## Official Script

HEX is a continuity layer for computer work.

It does not replace the browser, the file system, or the applications people already use. It floats alongside them and keeps the thread of the work intact while the user moves between them.

Modern work is fragmented. A person begins with an intention, opens tabs, switches applications, downloads files, looks up terms, and follows new ideas. The useful material spreads across the computer, while the reason it all belongs together becomes easier to lose.

HEX gives that work a persistent center: the Thread.

The Thread holds the user's current intention. Pins hold concepts and definitions. Links hold useful destinations. Files hold working material. Notes and open loops hold what still needs attention. These pieces remain attached to the Thread even when tabs close and applications change.

HEX stays available as a small desktop edge overlay. It is visible when needed and quiet when it is not. The user continues working in normal applications; HEX preserves orientation and makes the work easy to resume.

AI is not the product and HEX is not a chatbot. Intelligence is used only where it removes friction, such as turning a term into a clean definition or helping organize captured material. It should remain optional, contextual, and calm.

HEX is local-first because the user's working context belongs close to the user. Its purpose is not to demand more attention. Its purpose is to protect attention that has already been invested.

The governing idea is simple:

**Apps and tabs are temporary. The Thread persists.**

HEX keeps the thread of your work intact while you move across your computer.

## Product Filter

Before adding a feature, ask:

**Does this help the user preserve, understand, or resume the active Thread?**

If the answer is no, it probably does not belong in HEX.

## Memory Line

**HEX is the quiet layer that keeps my work together across tabs, apps, links, and files.**
